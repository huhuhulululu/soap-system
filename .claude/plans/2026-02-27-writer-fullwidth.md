# Writer 全宽单列重构

> **Goal:** 去掉左右分栏，改为全宽单列流（类似 Batch），表单紧凑，结果在下方。

## 结构变化

**当前：**
```
Header Bar
├── 左栏 (col-span-5)
│   ├── Step 1: 基础设置 / 患者信息 / 评估参数 / 主观必填
│   ├── Step 2: R项审核 (4个分组卡片)
│   └── Step 3: 警告 + 生成按钮
└── 右栏 (col-span-7)
    └── 结果列表
```

**目标：**
```
Header Bar (不变)
├── 表单区 (全宽, max-w-4xl)
│   ├── Card 1: 基础设置 + 患者信息 (紧凑网格)
│   ├── Card 2: 评估参数 + 主观必填 (紧凑网格)
│   ├── R项审核 (折叠面板, 默认收起)
│   ├── 警告 + 生成按钮
│   └── 进度条 (填写进度)
└── 结果区 (全宽)
    └── 结果列表 (与 Batch 相同的展开/折叠卡片)
```

## 关键改动

1. 删除 `grid grid-cols-1 lg:grid-cols-12` 和 `lg:col-span-5` / `lg:col-span-7`
2. 外层改为 `max-w-4xl mx-auto`
3. Step 1 的 4 个卡片合并为 2 个（基础+患者 / 评估+主观）
4. Step 2 R项审核改为折叠面板（默认收起，标题显示"R项审核 (4组)"，点击展开）
5. Step 3 的生成按钮和警告直接跟在表单后
6. 结果区从右栏移到下方，全宽显示
7. 去掉步骤编号圆圈（①②③），改为卡片标题
